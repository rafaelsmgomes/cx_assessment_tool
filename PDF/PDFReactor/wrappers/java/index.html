RealObjects PDFreactor Web Service Java API

In the subdirectories you'll find the following files:
  • lib\pdfreactor-wrapper.jar - The PDFreactor Java wrapper jar
  • sample\APISample.java - A Sample showing the simple integration of PDFreactor into Java applications

The API documentation can be found in the PDFreactor installation directory at doc/webservice/java.html.

Please see the PDFreactor manual for more information.
